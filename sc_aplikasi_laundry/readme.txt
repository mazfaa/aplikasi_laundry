Cara menjalankan program :

Unduh dan Instal Program Xampp dan Composer di Komputer.
Ekstrak file menggunakan program WinRAR.
Copy folder laundry, lalu copas ke folder htdocs.
Aktifkan Apache dan MySQL di Xampp.
Buka browser, lalu buka domisili localhost/phpMyAdmin.
Bikin database baru dengan nama laundry.
Import database program ke pada database laundry.
Buka terminal di pada project, ketikan php artisan serve.
Jalankan project dengan ketik http://127.0.0.1:8000/.
Login dengan email : admin, Password : rahasia.
Selesai.
